jquery.multiadder.js demo

jquery.multiadder.js provides a generic markup pattern to prototype "multi adders". Click the "add" buttton to add an option. An optional data attribute (data-maxrows) limits the amount.

This can be used independent of Twitter Bootstrap, just make to include the necessary .hide CSS class.

To run locally

  • Make sure you have nodejs and bower installed
  • Git clone the repository git clone git@github.com:Wolfr/jquery.bootstrap.simple.lightbox.git
  • Run npm install and bower install

To build

  • Make sure you have grunt installed
  • Run grunt
