jquery.multiadder.js demo

jquery.multiadder.js provides a generic markup pattern to prototype "multi adders". Click the "add" buttton to add an option. A data attribute (data-maxrows) limits the amount.

This can be used independent of Twitter Bootstrap, just make to include the necessary .hide CSS class.
